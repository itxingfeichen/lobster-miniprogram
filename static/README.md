# 静态资源说明

## 📁 目录结构

```
static/
├── images/           # 图片资源
│   ├── logo.png     # 应用 Logo（建议尺寸：200x200px）
│   └── default-avatar.png  # 默认头像（建议尺寸：200x200px）
│
└── icons/            # TabBar 图标
    ├── home.png          # 首页图标（未选中）81x81px
    ├── home-active.png   # 首页图标（选中）81x81px
    ├── learn.png         # 学习图标（未选中）81x81px
    ├── learn-active.png  # 学习图标（选中）81x81px
    ├── code.png          # 代码图标（未选中）81x81px
    ├── code-active.png   # 代码图标（选中）81x81px
    ├── mine.png          # 我的图标（未选中）81x81px
    └── mine-active.png   # 我的图标（选中）81x81px
```

## 🖼️ 图标规范

### TabBar 图标要求

| 属性 | 要求 |
|------|------|
| 尺寸 | 81x81 像素（推荐） |
| 格式 | PNG（支持透明背景） |
| 大小 | 单文件 < 40KB |
| 颜色 | 未选中：#7A7E83，选中：#3cc51f |

### 应用 Logo

| 属性 | 要求 |
|------|------|
| 尺寸 | 200x200 像素 |
| 格式 | PNG |
| 大小 | < 200KB |
| 用途 | 首页展示、关于页面 |

### 默认头像

| 属性 | 要求 |
|------|------|
| 尺寸 | 200x200 像素 |
| 格式 | PNG（圆形或圆角） |
| 大小 | < 100KB |
| 用途 | 个人中心默认头像 |

## 🎨 设计资源

### 颜色规范

- **主色调**: #3cc51f（微信绿）
- **文字色**: #333333（主要）、#666666（次要）、#999999（辅助）
- **背景色**: #F8F8F8（页面）、#FFFFFF（卡片）

### 图标风格

- 线性图标或面性图标（保持统一）
- 简洁现代风格
- 圆角设计（与整体 UI 一致）

## 📥 资源获取

### 推荐工具

- **图标制作**: [IconFont](https://www.iconfont.cn/)、[Figma](https://www.figma.com/)
- **图片压缩**: [TinyPNG](https://tinypng.com/)
- **在线设计**: [Canva](https://www.canva.com/)

### 临时占位

开发阶段可以使用以下占位方案：

1. 使用 emoji 作为临时图标
2. 使用在线占位图服务
3. 使用设计工具快速生成简单图标

## ⚠️ 注意事项

1. **小程序包大小限制**: 主包限制 2MB，注意控制图片大小
2. **图片格式**: 优先使用 PNG（透明背景）、JPG（照片）
3. **命名规范**: 使用小写字母和中划线，例：`home-active.png`
4. **多端适配**: 确保图标在各平台小程序中显示正常

## 🔄 资源更新

如需更新资源：

1. 替换对应文件
2. 清除编译缓存
3. 重新运行项目
4. 在真机上测试显示效果
